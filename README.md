##  Brain Games
Is a set of five console games built on the principle of popular mobile applications for brain pumping. Each game asks questions that need to be answered correctly. After three correct answers, it is considered that the game is over. Incorrect answers end the game and offer to go through it again. Games:

* Calculator. Arithmetic expressions that need to be calculated.
* Progression. Search for missing numbers in a sequence of numbers.
* Definition of an even number.
* Determination of the greatest common divisor.
* Definition of a prime number.
